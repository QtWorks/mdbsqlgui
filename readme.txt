MdbSqlGUI - MariaDB SQL Gui mail:jkam.mail@gmail.com
----------------------------------------------------

Project goals:
--------------
- Simple GUI frontend for maintaining MariaDB databases
- Qt5 based without other dependencies
- GPL license

Features:
---------
- Minimalistic GUI
- Connection management
- Tab based view
- Database management
- User management user/host/database grants
- queries with saving
- direct editing data in tables
- table management with templates support
