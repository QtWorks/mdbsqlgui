MdbSqlGUI - MariaDB SQL Gui
---------------------------

Project goals:
--------------
- Simple GUI frontend for maintaining MariaDB databases
- Qt5 based without other dependencies
- GPL license

Features: please see change.log file for current state of features
---------
- Minimalistic GUI
- Connection management
- Tab based view
- Database management
- User management user/host/database grants
- queries with saving
- direct editing data in tables
- table management with templates support

Install on blackPanther OS (www.blackpanther.hu)
---------[in console]--------
Update repository first: updating repos

Install appliction: installing mdbsqlgui

Start from menu: MDBSqlGui
